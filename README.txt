🎛 RGB CONTROL SYSTEM - By Furaiden

Ce ZIP contient tout ce qu’il te faut pour transformer une salle en show lumineux avec des téléphones !

📂 Fichiers inclus :
- admin.html : Interface de contrôle avec 64 couleurs, palette rapide et effets
- client.html : Interface des spectateurs, s'adapte à l'écran et reçoit les commandes
- server.js : Serveur Node.js WebSocket pour relier tout le monde
- README.txt : Ce fichier

🚀 Utilisation :
1. Lancer le serveur avec `node server.js`
2. Depuis le PC, aller sur : http://[IP]:3000/admin
3. Depuis chaque téléphone : http://[IP]:3000/client

🌐 Attention :
- Tous les appareils doivent être connectés au même réseau local
- Le PC peut faire hotspot s’il n’y a pas de routeur
